# TripGenie — Landing Page

Page d'accueil statique (HTML/CSS, sans build ni script) pour [TripGenie](https://github.com/loties1533/tripgenie-app), le générateur d'itinéraires de voyage.

## Structure

- `index.html` — contenu de la page (hero, fonctionnalités, à propos).
- `css/style.css` — thème sombre, aucune animation.
- `assets/` — captures d'écran de l'application.

## Aperçu en local

```bash
python3 -m http.server 8000
# puis ouvrir http://localhost:8000
```
